## Vision

Create the best coaching platform in the world.

Not another tracker.

Not another wearable dashboard.

A platform that helps people achieve real results.

---

## Philosophy

Data should serve coaching.

Coaching should serve people.

Technology should disappear.

---

## Core Principles

### 1. Automate Everything

Users should never enter data twice.

Import whenever possible.

Read from devices whenever possible.

Reduce friction.

### 2. Explain Everything

No mystery scores.

Every recommendation should explain **WHY**.

### 3. Trends Beat Snapshots

Weekly trends.

Monthly trends.

Behavior over time.

Avoid obsessing over daily fluctuations.

### 4. Coach First

AI never replaces coaches.

AI amplifies coaches.

### 5. Transparency

Show users the data.

Then explain the data.

### 6. Simplicity

If a feature requires a tutorial...

It is too complicated.

---

## Every Feature Must Answer

- Does this save time?
- Does this improve adherence?
- Does this improve coaching?
- Does this reduce friction?
- Would users miss this if removed?
